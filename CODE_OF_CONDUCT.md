# Code of Conduct

## Expected Behavior

- Be direct and respectful
- Focus on technical clarity
- Prefer evidence over assertion
- Keep disagreement factual and specific

## Unacceptable Behavior

- Harassment or abuse
- Personal attacks
- Deliberate misrepresentation of verification or support status
- Spam, trolling, or repeated off-topic disruption

## Reporting

Open a private maintainer contact channel if one exists. If not, open a GitHub issue only when the report can be made safely and without exposing private information.
