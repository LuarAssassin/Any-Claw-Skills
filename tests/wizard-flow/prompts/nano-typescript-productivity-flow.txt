# Wizard Flow Test: Nano TypeScript + Productivity

Walk through the wizard choosing:
- Reference mode: NanoClaw
- Tier: Nano
- Stack: TypeScript
- Providers: Anthropic
- Channels: Discord
- Domains: Productivity
- Options: None

## Expected Flow

1. Agent detects empty directory
2. Asks for project name -> "nano-bot"
3. Establishes the reference mode / product shape -> NanoClaw
4. Presents tiers -> choose Nano
5. Confirms TypeScript stack
6. Asks for providers -> select Anthropic
7. Asks for channels -> select Discord
8. Asks for domains -> select Productivity
9. Asks for options -> none
10. Confirms -> yes
11. Generates scaffold
12. Generates discord adapter
13. Generates anthropic provider
14. Generates productivity tools
15. Review + handoff

## Verification

- [ ] package.json exists
- [ ] src/index.ts exists
- [ ] src/channels/discord.ts exists
- [ ] src/provider.ts or src/providers/anthropic.ts exists
- [ ] src/tools/productivity.ts exists
- [ ] .env.example has ANTHROPIC_API_KEY, DISCORD_BOT_TOKEN
