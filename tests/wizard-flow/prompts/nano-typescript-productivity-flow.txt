# Wizard Flow Test: Nano TypeScript + Productivity

Walk through the wizard choosing:
- Tier: Nano

- Providers: Anthropic
- Channels: Discord

1. Agent detects empty directory
2. Asks for project name -> "nano-bot"
3. Presents tiers -> choose Nano
4. Confirms TypeScript stack

6. Asks for channels -> select Discord
7. Asks for domains -> select Productivity

10. Generates scaffold
11. Generates discord adapter
12. Generates anthropic provider
13. Generates productivity tools
14. Review + handoff
